# c2stem-server

A general server for c2stem Vue applications.

## Installation

- Clone Repository:

```sh
git clone git@github.com:c2stem/c2stem-server.git --recursive
```

- Front End:
Setup Vue static app.

```sh
cd client
npm install
npm run build
```

- Back End:
Setup Node Express server with MongoDB.

```sh
cd server
npm install
```

## Running locally

```sh
cd server
nodemon index.js
```
